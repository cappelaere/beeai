## Key Features

- **System-wide toggle** - Environment variable + user preference
- **Enhanced UI** - Larger fonts, better contrast, focus indicators
- **Text-to-Speech** - On-prem Piper TTS with MP3 output
- **Keyboard Navigation** - Full keyboard accessibility
- **Screen Reader Support** - ARIA labels and live regions
- **WCAG 2.1 Level AA** - Full compliance

## Command

```bash
/settings 508 on   # Enable Section 508 mode
/settings 508 off  # Disable Section 508 mode
```
