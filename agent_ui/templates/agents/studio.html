What Gets Created

When you create an agent, the following will be generated:

  • Agent Directory: agents/<agent_id>/
  • Configuration: agents/<agent_id>/agent.py
  • Skills Documentation: agents/<agent_id>/SKILLS.md (AI-generated)
  • Registry Entry: Added to agents.yaml

After Creation

Your agent will:

  • Appear in the Agent Collection
  • Be available in the chat agent selector
  • Have access to standard tools
  • Have AI-generated skills documentation

Tips

  • Choose a descriptive agent ID (can't be changed later)
  • Be specific in the skills prompt about the agent's domain expertise
  • Mention any specific tools or APIs the agent should use
  • You can edit the generated SKILLS.md file later
